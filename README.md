# URL Shortener com Django 1.9 e Python 2.7

Este Ã© um projeto de encurtador de URLs desenvolvido em Django 1.9 e Python 2.7. O projeto permite encurtar URLs longas e gerar QR codes para cada URL encurtada.

## Crie um ambiente virtual e o ative
``` console
foo@bar:~$ virtualenv venv
. venv/bin/activate
```


## Instale as dependencias
``` console
foo@bar:~$ pip install -r requirements.txt
```

## Rode o projeto
``` console
foo@bar:~$ python manage.py runserver &
```
